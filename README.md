# upi-fraud

This tiny project folder contains a Python virtual environment and git initialization.

Getting started

- Create and activate the virtual environment (already created as `.venv`):

  - macOS / Linux:

    ```bash
    source .venv/bin/activate
    ```

  - Windows (PowerShell):

    ```powershell
    .venv\Scripts\Activate.ps1
    ```

  - Windows (cmd.exe):

    ```cmd
    .venv\Scripts\activate.bat
    ```

- Install dependencies using `pip install -r requirements.txt` (create this file if needed).
- To open the project in VS Code from a local machine, run `code .` inside the `upi-fraud` folder.

Files of interest

- `.venv/` — Python virtual environment (ignored by `.gitignore`)

Next steps

- Add a `requirements.txt` and your project sources.
- Optionally create a `pyproject.toml` or `setup.cfg` for packaging.
